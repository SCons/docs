SCons API Documentation

Attention

This is the internal API Documentation for SCons. The documentation is automatically generated for each release from the source code using the Sphinx tool. Missing information is due to shortcomings in the docstrings in the code, which are by no means complete (contributions welcomed!).

The target audience is developers working on SCons itself: what is “Public API” is not clearly deliniated here. The interfaces available for use in SCons configuration scripts, which have a consistency guarantee, are those documented in the SCons Reference Manual.
