23.7. Checking for the Presence of a program

Check for the presence of a program by using the CheckProg method: 

env = Environment()
conf = Configure(env)
if not conf.CheckProg('foobar'):
  print('Unable to find the program foobar on the system')
  Exit(1)
env = conf.Finish()
