12.4. Updating the Modification Time of a File: The Touch Factory

If you just need to update the recorded modification time for a file, use the Touch factory: 

      Command("file.out", "file.in",
              [
                Copy("$TARGET", "$SOURCE"),
                Touch("$TARGET"),
              ])

Which executes as: 

      % scons -Q
      Copy("file.out", "file.in")
      Touch("file.out")
